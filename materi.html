Struktur Data Html


Setiap halaman HTML setidaknya memiliki struktur dasar yang terdiri dari : Tag DTD atau DOCTYPE, tag html, tag head, dan tag body. Inilah yang merupakan struktur paling dasar dari HTML, walaupun HTML tidak hanya berisi struktur tersebut.

  1. DOCTYPE

    Mendefinisikan versi HTML yang digunakan dalam dokumen. Pada HTML5, deklarasi ini adalah <!DOCTYPE html>.

  2. Elemet Html

    Elemen ini (<html>) adalah elemen root yang mengandung semua konten di halaman, termasuk elemen <head> dan <body>.

  3. Head Section

    Dibungkus dalam tag <head>, bagian ini biasanya berisi meta-informasi tentang dokumen, link ke stylesheet, dan lain-lain.

  4. Meta Tags

    Seperti <meta charset="UTF-8"> yang menentukan encoding karakter.

  5. Title

    Judul dokumen yang akan muncul pada tab browser, ditentukan dalam tag <title>.

  6. Body Section

    Dibungkus dalam tag <body>, bagian ini berisi semua konten yang akan ditampilkan di halaman web.

  7. Headings

    Elemen seperti <h1>, <h2>, <h3>, dan seterusnya digunakan untuk memberikan judul atau subjudul

  8. Paragraph

    Elemen <p> digunakan untuk teks biasa.

  9. Hyperlinks

    Elemen <a> digunakan untuk tautan

  10. Lists

    Elemen <ul> untuk unordered list dan <ol> untuk ordered list

  11. Images

    Elemen <img> untuk menampilkan gambar

Versi Html


Hingga saat ini sudah 5 versi yang anda kenal,seperti berikut ini :

  1. HTML v1.0 Versi pertama dari HTML ini adalah pionir dari perkembangan HTML. Tidak heran jika versi ini memiliki banyak sekali kelemahan termasuk tampilannya yang masih sangat sederhana. HTML versi 1.0 ini sudah mampu mendukung peletakan image pada dokumen tanpa adanya wrapping, heading, hypertext, paragraph, cetak tebal dan miring pada penulisan text.
  2. HTML v2.0 (24 November 1995) HTML versi 2.0 adalah pionir dari web interaktif seperti yang Anda temukan saat ini. Dibandingkan dengan versi pertama, struktur HTML lebih tertata rapi dan mampu menampilkan form dokumen. Dengan adanya form tersebut, Anda bisa memasukkan alamat, nama, saran dan kritik pada suatu dokumen.
  3. HTML v3.0 Jika dibandingkan dengan kedua versi HTML sebelumnya, HTML v3.0 tidak memiliki umur yang lama. Itu disebabkan karena ada versi terbaru dari versi 3, yaitu versi 3.2. Di versi 3.0, HTML sudah bisa dipergunakan untuk meletakkan tabel dan gambar. Fitur terbaru dari HTML 3.0 juga mampu untuk mendukung penggunaan rumus matematika pada dokumennya.
  4. HTML v3.2 (14 Januari 1996) HTML versi ini adalah pembaruan dari versi 3.0. Hingga saat ini, jenis HTML ini adalah yang paling sering dipergunakan. Versi ini dipublikasikan dan versi pertama yang dikembangkan serta distandarisasi khusus oleh W3C. Versi 3.2 ini pada awalnya disebut dengan Wilbur sebelum dikenal dengan nama HTML versi 3.2. Fitur yang ada di versi ini diantaranya adalah gambar untuk background, tabel, style, frame, hingga teks di sekeliling gambar. Jika di versi sebelumnya, Anda hanya bisa menggunakan HTML saja untuk pembuatan dokumen namun di versi ini tidak. Artinya, Anda bisa menggunakan script di luar HTML untuk kinerja HTML yang lebih baik atau untuk tujuan tertentu. Beberapa script HTML yang bisa dipergunakan diantaranya adalah Javascript dan VBScript.
  5. HTML v4.0 (18 Desember 1997) Dibandingkan versi sebelumnya, versi ini hadir dengan banyak perubahan dari v3.2. Beberapa diantaranya ada di tabel, link, image, text, meta, form dan imagemaps.
  6. HTML v4.01 (5 Mei 2000) Setelah ada versi 4.0, terdapat versi 4.01 yang merupakan revisi dari versi sebelumnya. Di versi ini, ada perbaikan kesalahan minor (kecil). Dari struktur pada HTML yang ada di versi ini, membuat HTML v4.01 menjadi standarisasi elemen serta atribut script XHTML 1.0.
  7. HTML v5.0 (28 Oktober 2014) Bisa dikatakan, versi 5.0 dari HTML adalah versi paling canggih dan paling stabil dibandingkan versi sebelumnya. Pembaharuan ini baru diperkenalkan pada tahun 2009. Versi terbaru ini dikembangkan lagi pada tanggal 4 Maret 2010 oleh W3C dan IETF (Internet Engineering Task Force). IETF sendiri merupakan organisasi yang sudah menangani HTML sejak v2.0.

Tag Html

  1. <html> Tag:

    Tag <html> adalah elemen utama dalam dokumen HTML. Ini menandakan awal dari dokumen HTML dan akhir dari dokumen HTML. Semua elemen HTML lainnya harus berada di dalam tag <html>. Biasanya, tag <html> memiliki dua atribut: lang (yang menentukan bahasa dokumen) dan dir (arah teks, seperti "ltr" untuk kiri ke kanan atau "rtl" untuk kanan ke kiri).


  2. <head> Tag:

    Tag <head> digunakan untuk menambahkan informasi meta dan elemen-elemen lain yang tidak akan ditampilkan di halaman web itu sendiri.Biasanya berisi elemen-elemen seperti <title>, <meta>, <link>, dan <script>. <title> adalah salah satu elemen penting di dalam <head> yang menentukan judul halaman web yang akan ditampilkan di tab browser.


  3. <title> Tag:

    Tag <title> digunakan untuk menentukan judul halaman web yang akan ditampilkan di browser ketika halaman web tersebut dibuka atau di-bookmark. Ini adalah elemen penting karena membantu dalam identifikasi konten halaman web.


  4. <body> Tag:

    Tag <body> digunakan untuk menampilkan semua konten yang akan ditampilkan di halaman web, seperti teks, gambar, video, dan elemen-elemen lainnya. Semua konten yang ditampilkan di halaman web biasanya ditempatkan di dalam tag <body>.

Tag Syntax Option

  1. <select> Element:

    Dalam elemen <select>, tag <option> digunakan untuk membuat pilihan dalam bentuk dropdown atau daftar yang dapat dipilih oleh pengguna. Pengguna dapat memilih satu atau lebih opsi dari daftar pilihan yang diberikan.


  2. <datalist> Element:

    Dalam elemen <datalist>, tag <option> digunakan untuk membuat daftar opsi yang dapat digunakan oleh elemen input <input> dengan atribut list.


  3. <optgroup> Element:

    Dalam elemen <optgroup>, tag <option> digunakan untuk mengelompokkan beberapa opsi bersama-sama dalam satu grup atau kategori tertentu. Ini berguna untuk membuat daftar yang lebih terstruktur.

Fungsi Berbagai Tag dalam HTML

Tag Deskripsi
<html> Tag utama yang menandai awal dan akhir dari dokumen HTML.
<head> Mengandung elemen-elemen yang tidak ditampilkan di halaman web seperti <title>, <meta>, <link>, dan <script>.
<title> Menentukan judul halaman web yang ditampilkan di tab browser.
<meta> Menyediakan informasi meta tentang dokumen HTML.
<link> Menghubungkan dokumen HTML ke berkas eksternal seperti CSS.
<script> Menyisipkan kode JavaScript ke dalam halaman web.
<body> Menampilkan semua konten yang terlihat di halaman web.
<h1> <h2> ... <h6> Tag-heading untuk membuat judul atau tajuk di halaman web.
<p> Menandai paragraf teks.
<a> Membuat tautan atau hyperlink.
<img> Menampilkan gambar di halaman web.
<ul> <ol> <li> Membuat daftar tak terurut dan terurut serta item dalam daftar.
<div> Wadah umum untuk mengelompokkan dan mengatur elemen HTML lainnya.
<form> Membuat formulir untuk mengirimkan data ke server.
<input> Mengumpulkan masukan dari pengguna dalam formulir.

Format Teks Dalam HTML

Tag Deskripsi
<b> format text bold/tebal
<i> format text italic/miring
<u> format text underline/garis bawah
<small> format text kecil
<strong> format text yang hampir sama dengan format bold
<sub> format subscripted teks
<sup> format superscripted teks
<ins> format text garis bawah
<del> format text dengan garis di tengah
<mark> format text yang berwarna
